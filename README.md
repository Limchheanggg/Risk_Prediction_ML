# 🧠 Risk Prediction ML

A Machine Learning project focused on predicting illness risk using a Logistic Regression model built from scratch using NumPy.

---

## 📌 Project Overview

This project demonstrates how Logistic Regression works internally without relying on machine learning libraries like Scikit-learn.

The model predicts illness risk based on lifestyle-related features such as:

- Age
- Sleep Duration
- Water Intake
- Stress Level
- Screen Time
